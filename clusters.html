Example 1: Agreements and Constitution

The following analysis uses peace agreements that fell between the Sudan constitutions of 2005 and 2019.

This is an example of a network built from provision-section pairs found by mapping agreements to the 2019 Sudan constitution. Provisions are represented as in pink nodes and are labelled with the agreement ID and provision number. Sections are represented as blue nodes and are labelled with the section number only.

The network comprises a set of disjoint subgraphs or clusters. There are 65 clusters in this example. The simplest cluster is a provision-section pair. Because some provisions map onto more than one section, and sections may be mapped by more than one provision, larger clusters can form as subsequent provision-section pairs join. There are 20 of these larger clusters in this example, the remaining 45 clusters are simple provision-section pairs.
